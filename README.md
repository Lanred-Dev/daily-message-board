# Daily Message Board

This project provides a simple hardware and software solution for displaying a daily message and image on a small oled screen. It is programmed to run on a ESP32 microcontroller and can be easily customized to display different messages and images.

### Requirements

- ESP32 microcontroller
- OLED display (SSD1306)
- Piezoelectric buzzer
- Button
- Switch
- RTC (DS3231)
